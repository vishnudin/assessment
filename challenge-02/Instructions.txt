Steps to run
************
Created Dockerfile and docker-compose.yml at assessment/challenge-02.

Prod Environment
================
cd assessment/challenge-02 
set ENV=PROD 
docker-compose up --build
docker ps
docker exec -it container_id /bin/sh
apk update 
apk add curl
curl http://localhost:8080
o/p: Hello there!  This is PROD Golang web server!
